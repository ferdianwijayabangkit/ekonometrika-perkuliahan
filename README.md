Pusat sumber daya digital resmi untuk mata kuliah Ekonometrika di Universitas Sultan Ageng Tirtayasa (UNTIRTA). Repositori ini berfungsi sebagai wadah penyimpanan materi teori, aplikasi statistik dalam ekonomi, serta referensi pendukung bagi mahasiswa.

## 🔗 **Tautan Referensi Utama**

Seluruh literatur pendukung, buku teks, dan materi referensi dapat diakses melalui tautan berikut:

**[Buka Kumpulan Ebook & Referensi](https://untirtaacid-my.sharepoint.com/my?id=%2Fpersonal%2Fferdian%5Fbangkit%5Funtirta%5Fac%5Fid%2FDocuments%2FUNTIRTA%2Febook%2FEkonometrika&ga=1)**

## ✨ Isi Repositori

- **Modul Kuliah**: Materi presentasi yang mencakup analisis regresi, asumsi klasik, model data panel, hingga analisis deret waktu (*time series*).
- **Panduan Analisis**: Tutorial penggunaan perangkat lunak statistik untuk pengolahan data ekonomi.
- **Silabus & RPS**: Rencana Pembelajaran Semester yang merinci target kompetensi dan jadwal pertemuan.
- **Dataset Latihan**: Kumpulan data ekonomi yang digunakan sebagai bahan praktik analisis mandiri maupun tugas.

## 🔧 Format & Alat Bantu

- **Dokumentasi Materi**: `PDF` / `PowerPoint`
- **Alat Analisis**: `R` / `Python` / `EViews` / `Stata`
- **Platform**: `GitHub`

## 🚀 Cara Mengakses Materi

### Prasyarat
- Koneksi internet untuk sinkronisasi data.
- Aplikasi pembaca PDF.
- Perangkat lunak statistik yang sesuai dengan instruksi di kelas.
- Akun Microsoft institusi untuk akses penuh ke SharePoint UNTIRTA.

### Langkah-langkah Penggunaan

1.  **Navigasi Materi**
    Cari folder berdasarkan topik pembahasan (misalnya: Regresi Linear Berganda atau Heteroskedastisitas).

2.  **Unduh Referensi**
    Pilih buku atau modul yang diperlukan melalui link OneDrive/SharePoint yang tersedia, kemudian simpan di perangkat lokal.

3.  **Praktik Mandiri**
    Gunakan dataset yang tersedia di repositori untuk mereplikasi analisis yang telah diajarkan di kelas.
